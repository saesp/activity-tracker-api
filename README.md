# Activity Tracker API (back-end)
RESTful Web API built with ASP.NET Core and Entity Framework Core to manage and analyze daily activities.

## 🚀 Features
- Full CRUD operations for activities
- SQL Server database integration
- Entity Framework Core (ORM)
- Relationship between Activities and Categories
- RESTful API design

## 🧱 Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- C#

## 📡 API Endpoints
- GET /api/activities
- GET /api/activities/{id}
- POST /api/activities
- PUT /api/activities/{id}
- DELETE /api/activities/{id}

## 🧠 Project Purpose
This project was built to practice full-stack backend development, focusing on REST APIs, database design, and data-driven applications.
